# Carrot

[README in English](README.md)
## 特性
- 基于ant-design的企业级服务前端组件库
- 简洁、易用、场景化

## 支持环境
- 现代浏览器和 IE9 及以上

## 安装
``` bash
npm install carrot
```

## 示例
``` jxs
import { DatePicker } from 'antd';
ReactDOM.render(<DatePicker />, mountNode);
```

## 本地开发
```bash
$ git clone -b package http://192.168.0.22/front-end/carrot.git
$ npm install

# 文档演示
$ npm run doc
```
打开浏览器访问 http://127.0.0.1:6060
